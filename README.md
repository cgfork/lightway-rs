# Installation

## Prerequirement

### Ubuntu

```
$ sudo apt-get install build-essential libssl-dev
```

## Debian

```
$ sudo apt-get install build-essential libssl-dev pkg-config
```

### MacOS

```
$ brew install openssl
```

## Cargo Install

```
$ cargo install --git https://github.com/cgfork/lightway-rs --features tls
```
